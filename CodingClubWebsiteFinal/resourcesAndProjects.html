Projects We Have Built:

Our club's calculator app project kicked off with a flurry of brainstorming, where we divided tasks based on our diverse skill sets. Some members focused on designing the user interface, while others delved into the core logic, implementing mathematical functions. We used a collaborative Git repository to manage our code, enabling us to work simultaneously and resolve conflicts efficiently.

Our coding club embarked on building a chat application, leveraging JavaScript throughout the stack. We divided responsibilities, with some members focusing on the backend to enable real-time messaging, while others developed the cross-platform frontend interface. We implemented a Git workflow for version control and collaborative coding. This project provided valuable experience in client-server architecture, real-time data handling, and the collaborative nature of software development.

Our coding club's Flappy Bird game project was a nostalgic and challenging endeavor. into teams, with some members focusing on the game logic using JavaScript and the Canvas API, handling bird physics, collision detection, and scorekeeping. Others worked on the visual elements, creating pixel art for the bird, pipes, and background, and implementing animations. This project honed our skills in game development fundamentals, while also fostering teamwork and creative problem-solving.
